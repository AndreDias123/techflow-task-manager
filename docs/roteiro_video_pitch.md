# Roteiro do vídeo pitch (até 4 minutos)

## 1. Abertura
Olá, este é o projeto TechFlow Task Manager, desenvolvido para simular um sistema de gerenciamento de tarefas com base em metodologias ágeis.

## 2. Problema de negócio
A solução atende uma startup de logística que precisa acompanhar o fluxo de trabalho, priorizar tarefas críticas e monitorar o andamento da equipe.

## 3. Metodologia ágil
Adotei uma abordagem híbrida entre Scrum e Kanban. O Scrum foi utilizado para organizar backlog e incrementos, enquanto o Kanban foi usado para o fluxo visual das tarefas no GitHub Projects.

## 4. Demonstração do sistema
Apresente a tela principal, mostre o cadastro, a edição, a exclusão e o acompanhamento de status das tarefas.

## 5. Testes e qualidade
Explique que o projeto possui testes automatizados com Pytest e integração contínua com GitHub Actions, garantindo maior confiabilidade.

## 6. Mudança de escopo
Informe que, durante o projeto, foi simulada a necessidade de adicionar filtro por prioridade, mostrando como a documentação e o backlog foram atualizados.

## 7. Fechamento
Conclua destacando que a Engenharia de Software foi essencial para organizar requisitos, modelar o sistema, versionar entregas, automatizar qualidade e controlar mudanças.
